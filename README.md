# first-contribution 🚀

A hands-on repository used during the **GitHub Crash Course** session. The goal is simple: make your first pull request by adding your name and a fun fact to the Contributors section below. 🎉

---

## How to contribute 🛠️

1. Clone this repository to your local machine
2. Create a new branch with your name (e.g. `add-maria-garcia`)
3. Add your name and a fun fact to the **Contributors** section below
4. Commit, push your branch, and open a pull request

Full step-by-step instructions are in the course guide.
